# FOR-EPIC UI & UX

## Goal

Create a clear, dark, data-driven interface inspired by the Forma mockup.

## Business value

The app must be useful at a glance. Diego should understand weekly status in seconds, not audit a spreadsheet like Hacienda.

## Scope

- Dashboard layout
- Navigation
- Dark theme
- Metric cards
- Charts
- Mobile-first key actions
- Agent-readable UI guidelines

## Initial stories

- FOR-080 Dashboard layout
- FOR-081 Dark theme tokens
- FOR-082 Navigation shell
- FOR-083 Metric card component
- FOR-084 Chart component patterns

## Acceptance criteria

- UI follows docs/ui-guidelines.md.
- Dashboard prioritizes status, trends and next action.
- Mobile flow supports today's plan, add measurement, mark training done and shopping checklist.

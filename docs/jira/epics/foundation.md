# FOR-EPIC Foundation

## Goal

Create the technical and organizational base for Forma.

## Business value

A stable foundation allows AI agents and human reviewers to work safely on small, reviewable increments.

## Scope

- Monorepo skeleton
- Backend bootstrap
- Frontend bootstrap
- Docker Compose
- PostgreSQL
- CI
- Documentation
- Initial dashboard shell

## Initial stories

- FOR-001 Bootstrap monorepo
- FOR-002 Spring Boot backend skeleton
- FOR-003 React frontend skeleton
- FOR-004 Docker Compose local environment
- FOR-005 PostgreSQL and Flyway setup
- FOR-006 GitHub Actions CI
- FOR-007 Health endpoint
- FOR-008 Initial dashboard shell

## Acceptance criteria

- Project can be cloned and run locally.
- Backend and frontend have independent build commands.
- PostgreSQL starts locally through Docker Compose.
- First CI workflow validates the project.
- README explains local development.

# FOR-EPIC Nutrition Planner

## Goal

Create flexible nutrition templates for running, strength and rest days.

## Business value

Forma should make nutrition practical for late summer training, avoiding heavy dinners after evening runs while still meeting protein and energy targets.

## Scope

- Nutrition day templates
- Meal templates
- Food catalog
- Macro calculations
- Late running day meal flow
- Whey as optional tool

## Initial stories

- FOR-030 Nutrition day templates
- FOR-031 Food catalog
- FOR-032 Macro calculator
- FOR-033 Late running day flow
- FOR-034 Protein target tracking

## Acceptance criteria

- User can select running, strength or rest day.
- App shows meals and macro totals.
- Running day moves carbs earlier and keeps dinner light.
- Whey is represented as optional recovery/protein support.

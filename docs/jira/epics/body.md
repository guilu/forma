# FOR-EPIC Body Composition

## Goal

Track body composition and weekly trends.

## Business value

Forma should detect whether Diego is improving body composition, not just changing body weight.

## Scope

- Manual body measurements
- Weight, body fat, BMI and lean mass
- Derived metrics
- Trend charts
- Weekly check-in
- Future Withings normalization target

## Initial stories

- FOR-010 Manual body measurements
- FOR-011 Dashboard body metric cards
- FOR-012 Body trend charts
- FOR-013 Weekly body check-in
- FOR-014 Derived body metrics

## Acceptance criteria

- User can add measurements manually.
- App calculates fat mass and lean mass.
- Dashboard shows latest metrics and trend direction.
- Measurements can later be imported from Withings without changing the core Body model.
